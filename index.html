Snowcone

an icon font set by Tangent Snowball

This font set has been lovingly designed by the Tangent Snowball team to be an easily adjustable set of icons to use on websites and web–apps. Using a font set ensures all icons are fully scalable and only need one header call. It is currently in its infancy but we'll be expanding it henceforth.

Fork it here

Support

It's early days yet, but currently Snowcone works on all browsers except IE6-7 (due to the fact they dont support :before). To make these browsers happy, a jQuery fix has been included.

Implementation

  1. Download the files from the repo
  2. Add the /fonts/ directory to your site's root
  3. Include script (if IE6–7 support is needed) and css dependancies as shown on the barebones demo.
  4. Included in snowcone.css are a set of classes that call each icon. To include an icon in your page, attach one of these classes, and the base icon class to any element (as shown below).
