Snowcone

an icon font set by Tangent Snowball

This font set has been lovingly designed by the Tangent Snowball team to be an easily adjustable set of icons to use on websites and web–apps. Using a font set ensures all icons are fully scalable and only need one header call. It is currently in its infancy but we'll be expanding it henceforth. Fork it here

Support

It's early days yet, but currently Snowcone works on all modern browsers. IE6-8 don't play nice, due to the fact they don't support the :before pseudo class, but we're working on a solution (suggestions welcome).

Implementation

  1. Clone the repo
  2. Switch to the gh-pages branch
  3. From here, grab the demo.css file, and the /fonts/ directory with its contents
  4. Include the /fonts/ directory within your website's directory
  5. Either include the snowcone.css file witin your website's directory, or copy and paste the rules within to your website's CSS file.
  6. In the CSS rules are a set of classes that call each icon. To include an icon in your page, attach one of these classes, and the base icon class to any element.
